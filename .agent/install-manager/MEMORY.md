# Install Manager Memory Index

This index tracks durable operational memory for the local install-manager agent.

## How to use

After an install, major configuration change, or repair:

1. Update or create a focused memory file in `memory/`
2. Add a one-line pointer here
3. Keep entries factual, concise, and current
4. Prefer updating an existing topic file over creating duplicates

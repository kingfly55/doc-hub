## Scope

Create `src/doc_hub/documents.py` with the `DocumentNode` dataclass and the pure-Python tree-building algorithm. This milestone contains NO database code — only in-memory data structures and algorithms that convert a list of chunks plus optional manifest sections into a `list[DocumentNode]`.

**Important context on `source_file` naming**: the markdown parser (`_builtins/parsers/markdown.py`) sets `source_file` to the flat filename (`md_file.name`), never a nested directory path. The llms_txt fetcher uses the `__`-encoded convention (e.g. `models__openai.md`). The local_dir fetcher only works with whatever flat filenames already exist on disk; it does not add hierarchy on its own. Tree building must therefore treat `source_file` as the only reliable input for path inference.

## Detailed Specification

### 1. `DocumentNode` dataclass

```python
from __future__ import annotations
from dataclasses import dataclass

@dataclass
class DocumentNode:
    """A node in the inter-document hierarchy tree."""
    doc_path: str
    title: str
    source_url: str = ""
    source_file: str = ""
    parent_path: str | None = None
    depth: int = 0
    sort_order: int = 0
    is_group: bool = False
    total_chars: int = 0
    section_count: int = 0
```

Notes:
- `doc_path` is the unique tree key used by the DB layer later.
- `source_file` is only populated for concrete document nodes, never for synthetic group nodes.
- `is_group=True` marks virtual path or section nodes that have no direct chunk content.

### 2. Helper: `_doc_path_from_source_file(source_file: str) -> str`

Convert double-underscore filenames back to path segments:
- `"models__openai.md"` → `"models/openai"`
- `"agents.md"` → `"agents"`
- `"api__base.md"` → `"api/base"`
- `"index.md"` → `"index"`

```python
def _doc_path_from_source_file(source_file: str) -> str:
    stem = source_file.removesuffix(".md")
    return stem.replace("__", "/")
```

### 3. Helper: `_humanize_path_segment(segment: str) -> str`

Convert a path segment into a human-readable title:
- `"getting-started"` → `"Getting Started"`
- `"openai"` → `"Openai"`
- `"api_reference"` → `"Api Reference"`

```python
def _humanize_path_segment(segment: str) -> str:
    return segment.replace("-", " ").replace("_", " ").title()
```

### 4. Helper: `_slugify(title: str) -> str`

Generate a safe section path segment from an llms.txt heading.

```python
import re

def _slugify(title: str) -> str:
    return re.sub(r"[^a-z0-9]+", "-", title.lower()).strip("-")
```

### 5. Helper: `_derive_title(source_file: str, chunks_by_file: dict[str, list[Chunk]]) -> str`

Derive a document title:
1. Look for the first chunk from this `source_file` with `heading_level == 1`
2. If found, use its `heading`
3. Otherwise, humanize the last path segment of the doc path

```python
def _derive_title(source_file: str, chunks_by_file: dict[str, list[Chunk]]) -> str:
    file_chunks = chunks_by_file.get(source_file, [])
    for chunk in file_chunks:
        if chunk.heading_level == 1:
            return chunk.heading
    doc_path = _doc_path_from_source_file(source_file)
    return _humanize_path_segment(doc_path.rsplit("/", 1)[-1])
```

### 6. Helper: `_iter_path_prefixes(doc_path: str) -> list[str]`

Return all parent path prefixes for a concrete path in top-down order.

Examples:
- `"api/models/openai"` → `['api', 'api/models']`
- `"agents"` → `[]`

This helper is used to create missing virtual group nodes before the concrete document node is emitted.

### 7. Core: `build_document_tree(chunks, manifest_sections=None) -> list[DocumentNode]`

The main entry point. Accepts:
- `chunks`: list of `Chunk` objects (from `doc_hub.parse.Chunk`)
- `manifest_sections`: optional `list[dict]` from Milestone 4. Each dict has `{"title": str, "heading_level": int, "urls": list[str]}`.

Returns a flat `list[DocumentNode]` representing the entire tree. The flat list must already be in preorder, so callers can render it without rebuilding parent/child relationships.

```python
from collections import defaultdict
from doc_hub.parse import Chunk

def build_document_tree(
    chunks: list[Chunk],
    manifest_sections: list[dict] | None = None,
) -> list[DocumentNode]:
    if not chunks:
        return []

    chunks_by_file: dict[str, list[Chunk]] = defaultdict(list)
    for chunk in chunks:
        chunks_by_file[chunk.source_file].append(chunk)

    file_order = list(dict.fromkeys(chunk.source_file for chunk in chunks))
    file_to_doc_path = {
        source_file: _doc_path_from_source_file(source_file)
        for source_file in file_order
    }
    file_to_stats = {
        source_file: (
            sum(chunk.char_count for chunk in file_chunks),
            len(file_chunks),
            file_chunks[0].source_url if file_chunks else "",
        )
        for source_file, file_chunks in chunks_by_file.items()
    }
    url_to_source_file: dict[str, str] = {}
    for source_file in file_order:
        for chunk in chunks_by_file[source_file]:
            if chunk.source_url and chunk.source_url not in url_to_source_file:
                url_to_source_file[chunk.source_url] = source_file

    nodes: list[DocumentNode] = []
    emitted: set[str] = set()
    sort_order = 0

    def emit_group(doc_path: str, title: str, parent_path: str | None, depth: int) -> None:
        nonlocal sort_order
        if doc_path in emitted:
            return
        nodes.append(DocumentNode(
            doc_path=doc_path,
            title=title,
            parent_path=parent_path,
            depth=depth,
            sort_order=sort_order,
            is_group=True,
        ))
        emitted.add(doc_path)
        sort_order += 1

    def emit_document(
        *,
        doc_path: str,
        source_file: str,
        source_url: str,
        parent_path: str | None,
        depth: int,
    ) -> None:
        nonlocal sort_order
        if doc_path in emitted:
            return
        total_chars, section_count, _ = file_to_stats[source_file]
        nodes.append(DocumentNode(
            doc_path=doc_path,
            title=_derive_title(source_file, chunks_by_file),
            source_url=source_url,
            source_file=source_file,
            parent_path=parent_path,
            depth=depth,
            sort_order=sort_order,
            is_group=False,
            total_chars=total_chars,
            section_count=section_count,
        ))
        emitted.add(doc_path)
        sort_order += 1

    def ensure_path_chain(doc_path: str, *, namespace_prefix: str | None = None) -> str | None:
        """Create any missing virtual ancestors and return the immediate parent path."""
        prefixes = _iter_path_prefixes(doc_path)
        parent_path = namespace_prefix
        for prefix in prefixes:
            full_path = f"{namespace_prefix}/{prefix}" if namespace_prefix else prefix
            if full_path not in emitted:
                depth = prefix.count("/") + (1 if namespace_prefix else 0)
                emit_group(
                    full_path,
                    _humanize_path_segment(prefix.rsplit("/", 1)[-1]),
                    parent_path,
                    depth,
                )
            parent_path = full_path
        return parent_path

    def emit_flat_document(source_file: str) -> None:
        doc_path = file_to_doc_path[source_file]
        parent_path = ensure_path_chain(doc_path)
        emit_document(
            doc_path=doc_path,
            source_file=source_file,
            source_url=file_to_stats[source_file][2],
            parent_path=parent_path,
            depth=doc_path.count("/"),
        )

    if manifest_sections and any(section.get("urls") for section in manifest_sections):
        assigned_files: set[str] = set()

        for section in manifest_sections:
            section_title = section["title"]
            section_urls = section.get("urls", [])
            if not section_title:
                for url in section_urls:
                    source_file = url_to_source_file.get(url)
                    if source_file and source_file not in assigned_files:
                        assigned_files.add(source_file)
                        emit_flat_document(source_file)
                continue

            section_slug = _slugify(section_title)
            section_path = f"_section/{section_slug}"
            emit_group(section_path, section_title, None, 0)

            for url in section_urls:
                source_file = url_to_source_file.get(url)
                if not source_file or source_file in assigned_files:
                    continue
                assigned_files.add(source_file)
                doc_path = file_to_doc_path[source_file]
                parent_path = ensure_path_chain(doc_path, namespace_prefix=section_path)
                emit_document(
                    doc_path=f"{section_path}/{doc_path}",
                    source_file=source_file,
                    source_url=file_to_stats[source_file][2],
                    parent_path=parent_path,
                    depth=doc_path.count("/") + 1,
                )

        for source_file in file_order:
            if source_file not in assigned_files:
                emit_flat_document(source_file)
    else:
        for source_file in file_order:
            emit_flat_document(source_file)

    return nodes
```

Implementation notes:
- Preorder order is the actual emission order; do **not** sort the final list by `(depth, sort_order)`.
- `sort_order` must increase every time any node is emitted, including section groups and virtual groups.
- `depth` is the actual tree depth relative to the browse root: section roots and file roots are depth 0; children increment by 1 per level.
- The section namespace prefix (`_section/{slug}`) prevents path collisions between sections that contain the same document path.
- When a section is present, concrete document nodes keep their original `source_file` so the DB layer can link chunks by exact filename later.

## Success Criteria

1. `DocumentNode` is importable from `doc_hub.documents`.
2. `_doc_path_from_source_file()` correctly converts `__` filenames to `/` paths.
3. `_humanize_path_segment()` and `_slugify()` behave deterministically for simple filenames and section titles.
4. `build_document_tree([])` returns `[]`.
5. `build_document_tree()` produces a flat preorder list of `DocumentNode` objects from chunks with no manifest sections.
6. `build_document_tree()` produces grouped hierarchy when `manifest_sections` is provided, with documents namespaced under `_section/{slug}/` paths.
7. Virtual group nodes are created for intermediate paths with no content and are marked `is_group=True`.
8. `depth`, `sort_order`, `parent_path`, `total_chars`, and `section_count` are consistent with the emitted tree.
9. Files not matched by any manifest section are still included as root-level nodes.
10. Output order is preorder and must be consumed as-is; no caller should re-sort it by depth.

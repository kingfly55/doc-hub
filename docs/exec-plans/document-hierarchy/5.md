## Detailed Specification

### 1. New function: `run_build_tree(corpus, pool=None)`

```python
async def run_build_tree(
    corpus: Corpus,
    *,
    pool: asyncpg.Pool | None = None,
) -> dict[str, int]:
    """Build the document hierarchy tree for a corpus.

    Reads chunks and manifest from disk, calls build_document_tree(),
    then persists the tree via upsert_documents() + link_chunks_to_documents().

    This runs after run_index() in the full pipeline, or standalone via
    ``--stage tree``.
    """
```

Implementation requirements:
- Load chunks from `chunks_dir(corpus) / "chunks.jsonl"`.
- Load `manifest.json` from `raw_dir(corpus)` if it exists.
- Read `manifest_data.get("sections")`; pass `None` when the key is missing.
- Build the tree with `build_document_tree(chunks, manifest_sections=manifest_sections)`.
- Open a DB pool if one is not supplied.
- Always call `ensure_schema(pool)` before persistence so the tree stage can backfill an older corpus after the schema lands.
- Upsert nodes, link chunks, and delete stale documents in that order.
- Return a dict with keys `documents`, `linked_chunks`, and `deleted`.
- If `chunks.jsonl` is missing, return `{"documents": 0, "linked_chunks": 0, "deleted": 0}` without raising.
- If the tree is empty, return the same zero dict rather than touching the DB.

### 2. Update `run_pipeline()` — add tree stage after index

In the stage dispatch section of `run_pipeline()`, insert the tree stage block **after** the index stage block and **before** the unknown-stage error check.

Tree stage behavior:
- When `stage == "tree"`, skip fetch/parse/embed/index and run only `run_build_tree()`.
- When `stage is None`, run the full pipeline and then run the tree stage after `run_index()` using the same in-memory `pool` object passed into `run_pipeline()`.
- If `stage == "index"`, keep the existing return behavior unchanged.

Important: do not make the tree stage depend on re-running parse or embed. It must consume `chunks.jsonl` produced earlier and the manifest on disk.

Suggested block:

```python
    if stage == "tree":
        await run_build_tree(corpus, pool=pool)
        _log_elapsed(corpus, pipeline_start)
        return None

    if stage is None:
        result = await run_index(...)
        await run_build_tree(corpus, pool=pool)
```

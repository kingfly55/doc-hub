## Detailed Specification

### 1. New `_DOCUMENTS_DDL` constant in `db.py`

Add after `_META_DDL`:

```python
_DOCUMENTS_DDL = """
CREATE TABLE IF NOT EXISTS doc_documents (
    id            serial PRIMARY KEY,
    corpus_id     text NOT NULL REFERENCES doc_corpora(slug) ON DELETE CASCADE,
    doc_path      text NOT NULL,
    title         text NOT NULL,
    source_url    text NOT NULL DEFAULT '',
    source_file   text NOT NULL DEFAULT '',
    parent_id     int REFERENCES doc_documents(id) ON DELETE SET NULL,
    depth         smallint NOT NULL DEFAULT 0,
    sort_order    int NOT NULL DEFAULT 0,
    is_group      boolean NOT NULL DEFAULT false,
    total_chars   int NOT NULL DEFAULT 0,
    section_count int NOT NULL DEFAULT 0,
    UNIQUE (corpus_id, doc_path)
)
"""
```

Notes:
- `parent_id` must be nullable so virtual group nodes can be rooted and stale parent rows can be deleted without cascading into current descendants.
- `doc_path` remains the corpus-unique identity for document browsing.
- `source_file` is persisted for concrete document rows so chunk linking can use exact filename matching later.

### 2. New `_DOCUMENTS_INDEXES_DDL` constant

```python
_DOCUMENTS_INDEXES_DDL = """
CREATE INDEX IF NOT EXISTS doc_documents_corpus_id_idx
    ON doc_documents (corpus_id);

CREATE INDEX IF NOT EXISTS doc_documents_parent_id_idx
    ON doc_documents (parent_id);

CREATE INDEX IF NOT EXISTS doc_documents_corpus_sort_order_idx
    ON doc_documents (corpus_id, sort_order);

CREATE INDEX IF NOT EXISTS doc_documents_corpus_path_idx
    ON doc_documents (corpus_id, doc_path text_pattern_ops);
"""
```

Notes:
- Browsing queries order by `sort_order`, so the index must include `sort_order` directly.
- `doc_documents_corpus_path_idx` is for prefix subtree lookups (`path` + descendants).
- The `UNIQUE (corpus_id, doc_path)` constraint already creates a supporting index for exact lookups.

### 3. `_CHUNKS_DOCUMENT_ID_DDL` constant

Add `document_id` FK column to `doc_chunks` (idempotent ALTER):

```python
_CHUNKS_DOCUMENT_ID_DDL = """
ALTER TABLE doc_chunks ADD COLUMN IF NOT EXISTS document_id int
    REFERENCES doc_documents(id) ON DELETE SET NULL
"""

_CHUNKS_DOCUMENT_ID_INDEX = """
CREATE INDEX IF NOT EXISTS doc_chunks_document_id_idx
    ON doc_chunks (document_id)
"""
```

### 5. Ordering constraint

`_DOCUMENTS_DDL` MUST execute **after** `_CORPORA_DDL` (it references `doc_corpora(slug)`) and **before** `_CHUNKS_DOCUMENT_ID_DDL` (which references `doc_documents(id)`). Both must execute before the new index DDL statements.

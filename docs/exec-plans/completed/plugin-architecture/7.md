# Milestone 7 — Embedder Pluggability

**Status**: complete

## Scope

Make the embedding stage pluggable by extracting the Gemini-specific API code into a built-in embedder plugin and refactoring `embed.py` into a core orchestration module that works with any `Embedder` protocol implementation.

The core embed pipeline retains responsibility for:
- Caching (keyed by `content_hash + model_name + dimensions`)
- L2 normalization of all output vectors
- Batch orchestration and rate-limit pacing
- Writing `embedded_chunks.jsonl`

The embedder plugin is responsible for:
- Calling its specific API (Gemini, OpenAI, local model, etc.)
- Reporting its `model_name`, `dimensions`, `task_type_document`, `task_type_query`
- Handling API-specific retry logic within `embed_batch()` and `embed_query()`

This milestone also wires up dimension validation: the core pipeline checks that the embedder's `dimensions` property matches `DOC_HUB_VECTOR_DIM` (from Milestone 4) and fails fast if they differ.

```python
"""Built-in Gemini embedder plugin for doc-hub.

Uses the Google Generative AI (Gemini) API for text embedding.
Model: gemini-embedding-001 (768 dimensions by default).

Entry point name: "gemini"

Environment:
    GEMINI_API_KEY — required. Get a free key at https://aistudio.google.com/apikey

Rate limits (free tier):
    - 100 requests/minute
    - ~1000 requests per rolling window (~85 min)

The plugin handles retry logic for rate limit (429) and server (503)
errors with smart discrimination between per-minute and per-day quotas.
"""
from __future__ import annotations

import asyncio
import logging
import os
import random
from typing import Any

log = logging.getLogger(__name__)

# Rate limit wait times (Gemini free tier)
_PER_MINUTE_WAIT = 65.0   # per-minute quota
_PER_DAY_WAIT = 300.0     # per-day/rolling quota

_DEFAULT_MODEL = "gemini-embedding-001"
_DEFAULT_DIM = 768
_DEFAULT_MAX_RETRIES = 20


class GeminiEmbedder:
    """Embedder plugin using the Gemini embedding API.

    Entry point name: "gemini"

    Configuration via environment variables:
        GEMINI_API_KEY: API key (required)
        GEMINI_EMBEDDING_MODEL: Model name (default: gemini-embedding-001)
        GEMINI_EMBEDDING_DIM: Output dimensions (default: 768)

    The client is created lazily on first use to avoid import-time
    side effects and to respect env vars set after import.
    """

    def __init__(self) -> None:
        self._client: Any = None
        self._model = os.environ.get("GEMINI_EMBEDDING_MODEL", _DEFAULT_MODEL)
        self._dimensions = int(os.environ.get("GEMINI_EMBEDDING_DIM", str(_DEFAULT_DIM)))
        self._max_retries = _DEFAULT_MAX_RETRIES

    def _get_client(self) -> Any:
        """Lazily create the Gemini client."""
        if self._client is None:
            api_key = os.environ.get("GEMINI_API_KEY")
            if not api_key:
                raise RuntimeError(
                    "GEMINI_API_KEY environment variable not set. "
                    "Get a free key at https://aistudio.google.com/apikey"
                )
            import google.genai as genai  # type: ignore[import]
            self._client = genai.Client(api_key=api_key)
        return self._client

    @property
    def model_name(self) -> str:
        return self._model

    @property
    def dimensions(self) -> int:
        return self._dimensions

    @property
    def task_type_document(self) -> str:
        return "RETRIEVAL_DOCUMENT"

    @property
    def task_type_query(self) -> str:
        return "RETRIEVAL_QUERY"

    async def embed_batch(self, texts: list[str]) -> list[list[float]]:
        """Embed a batch of texts using the Gemini API.

        Handles retry logic with smart rate-limit discrimination:
        - PerMinute (429): wait 65s
        - PerDay (429): wait 300s
        - Other errors: exponential backoff

        Args:
            texts: List of text strings to embed.

        Returns:
            List of raw embedding vectors (NOT L2-normalized).
        """
        from google.genai import types as genai_types  # type: ignore[import]

        client = self._get_client()
        for attempt in range(self._max_retries):
            try:
                response = await asyncio.to_thread(
                    client.models.embed_content,
                    model=f"models/{self._model}",
                    contents=texts,
                    config=genai_types.EmbedContentConfig(
                        task_type=self.task_type_document,
                        output_dimensionality=self._dimensions,
                    ),
                )
                return [emb.values for emb in response.embeddings]
            except Exception as e:
                if attempt == self._max_retries - 1:
                    raise
                wait = self._compute_wait(e, attempt)
                log.warning(
                    "Embed batch attempt %d/%d failed: %s. Retrying in %.1fs",
                    attempt + 1, self._max_retries, e, wait,
                )
                await asyncio.sleep(wait)

        raise RuntimeError("embed_batch: unreachable")

    async def embed_query(self, query: str) -> list[float]:
        """Embed a single query using RETRIEVAL_QUERY task type.

        Uses simpler retry logic (4 attempts, exponential backoff)
        since queries are single requests, not batch operations.

        Args:
            query: Search query string.

        Returns:
            Raw embedding vector (NOT L2-normalized).
        """
        from google.genai import types as genai_types  # type: ignore[import]

        client = self._get_client()
        max_attempts = 4
        for attempt in range(max_attempts):
            try:
                resp = await client.aio.models.embed_content(
                    model=f"models/{self._model}",
                    contents=[query],
                    config=genai_types.EmbedContentConfig(
                        task_type=self.task_type_query,
                        output_dimensionality=self._dimensions,
                    ),
                )
                if not resp.embeddings:
                    raise RuntimeError(
                        f"Gemini returned empty embeddings for query: {query!r}"
                    )
                return resp.embeddings[0].values
            except Exception as e:
                if attempt == max_attempts - 1:
                    raise
                err_str = str(e)
                is_retryable = any(
                    s in err_str for s in
                    ("429", "503", "RESOURCE_EXHAUSTED", "ServiceUnavailable")
                )
                if is_retryable:
                    wait = 1 * (2 ** attempt)
                    log.warning(
                        "Embed query attempt %d/%d failed: %s. Retrying in %ds",
                        attempt + 1, max_attempts, e, wait,
                    )
                    await asyncio.sleep(wait)
                else:
                    raise

        raise RuntimeError("embed_query: unreachable")

    @staticmethod
    def _compute_wait(exc: Exception, attempt: int) -> float:
        """Compute retry wait time based on error type."""
        err_str = str(exc)
        if "429" in err_str or "RESOURCE_EXHAUSTED" in err_str:
            if "PerDay" in err_str:
                return _PER_DAY_WAIT
            return _PER_MINUTE_WAIT
        return (2 ** attempt) + random.uniform(0, 1)
```

```python
async def _embed_query_async(
    query: str,
    embedder: Embedder | None = None,
) -> list[float]:
    """Embed a query string using the embedder plugin.

    If no embedder is provided, resolves the default embedder from
    the plugin registry. The default is determined by reading the
    first available embedder, or "gemini" if available.

    IMPORTANT: Cross-corpus search only works correctly when all corpora
    use the same embedder. The query is embedded once with this embedder,
    and the resulting vector is compared against all corpora's chunk
    embeddings. If corpus A uses a different embedder than this one,
    similarity scores for corpus A will be meaningless.
    """
    from doc_hub.embed import l2_normalize

    if embedder is None:
        from doc_hub.discovery import get_registry
        registry = get_registry()
        # Default to "gemini" if available, otherwise use the first registered embedder.
        # This avoids a hard dependency on the Gemini plugin being installed.
        available = registry.list_embedders()
        if not available:
            raise RuntimeError(
                "No embedder plugins are registered. Install an embedder plugin "
                "(e.g. doc-hub ships with 'gemini' by default)."
            )
        default_name = "gemini" if "gemini" in available else available[0]
        embedder = registry.get_embedder(default_name)

    raw_vec = await embedder.embed_query(query)
    return l2_normalize(raw_vec)
```

**Why not hardcode "gemini":** If someone replaces the default Gemini embedder with an OpenAI embedder and has no `google-genai` installed, a hardcoded `get_embedder("gemini")` would fail. The fallback to the first available embedder makes the system more resilient to custom configurations.

## Design Decisions

1. **Lazy client creation**: `GeminiEmbedder.__init__()` does not create the API client. The client is created on first `embed_batch()` or `embed_query()` call. This avoids import-time side effects and allows env vars to be set after the plugin is instantiated.

2. **Retry logic stays in the plugin**: Different embedding APIs have different rate limits and error codes. The retry logic is inherently API-specific, so it belongs in the plugin, not the core.

3. **L2 normalization stays in the core**: All vectors must be L2-normalized before storage. This is a universal requirement for cosine distance to work correctly, so it's the core pipeline's responsibility.

4. **`inter_batch_sleep` as a parameter**: The 65-second sleep between batches is Gemini-specific (free tier: 100 RPM). Making it a parameter allows different embedders to have different pacing. The default of 65.0 is conservative — it's the Gemini free-tier value. The pipeline does NOT currently read this from corpus config or the embedder — it uses the default. If a user switches to a paid API or local model, they must either:
   - Pass `inter_batch_sleep=0` programmatically when calling `embed_chunks()`, or
   - Override via a `DOC_HUB_EMBED_SLEEP` env var (which should be added to `embed_chunks()` as: `inter_batch_sleep = float(os.getenv("DOC_HUB_EMBED_SLEEP", str(inter_batch_sleep)))`)

   **Decision: Add `DOC_HUB_EMBED_SLEEP` env var** to allow overriding the inter-batch sleep without code changes. Default 65.0 for Gemini free tier. The env var is read inside `embed_chunks()`:

   ```python
   inter_batch_sleep = float(os.getenv("DOC_HUB_EMBED_SLEEP", str(inter_batch_sleep)))
   ```

   **Warning for plugin authors:** The 65-second default is extremely conservative (Gemini free-tier specific). If you're using a paid API or local model, set `DOC_HUB_EMBED_SLEEP=0` or `DOC_HUB_EMBED_SLEEP=1` to avoid unnecessary delays. Document this prominently in the plugin author guide (M8).

5. **Same embedder for cross-corpus search**: Different embedding models produce incompatible vector spaces. All corpora in a deployment MUST use the same embedder for cross-corpus search to produce meaningful results. The `embedder` column allows per-corpus configuration, but this is intended for deployment-level standardization (e.g., switching all corpora from Gemini to OpenAI), not for mixing embedders within a deployment.

6. **`google-genai` remains a hard dependency for now**: The Gemini embedder's lazy import (`import google.genai` inside `_get_client()`) means the dependency isn't needed at import time — only when the Gemini embedder is actually used. However, `google-genai` is still listed as a runtime dependency in `pyproject.toml`. Making it optional (extras) would require `pip install doc-hub[gemini]` syntax, which adds complexity for the common case. Since Gemini is the only built-in embedder and is free, keeping it as a hard dependency is the right trade-off. If a second embedder is added (e.g., OpenAI), revisit this decision then.
